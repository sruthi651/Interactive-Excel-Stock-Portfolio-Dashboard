# 📊Interactive-Excel-Stock-Portfolio-Dashboard

An interactive **Stock Market Portfolio Dashboard** built in **Microsoft Excel** using **VBA** and **Power Pivot** to analyze investment performance, visualize portfolio data, and provide actionable insights through an intuitive dashboard.

---

# 📖 About the Project

Managing investment portfolios becomes easier when data is presented visually. This project demonstrates how Microsoft Excel can be transformed into a powerful business intelligence tool using advanced Excel features.

The dashboard allows users to input stock portfolio data, automatically process it using Power Pivot and VBA, and present meaningful insights through interactive charts, KPIs, and slicers.

This project was developed to showcase my Excel, data analysis, and dashboard development skills for **Business Analyst** and **Data Analyst** roles.

# ▶️ Getting Started

1. Download the Excel workbook.
2. Open it in Microsoft Excel (Desktop Version).
3. Enable Macros when prompted.
4. Use the **User Input** sheet to enter portfolio data.
5. Navigate to the **Dashboard** sheet to explore interactive reports and insights.
